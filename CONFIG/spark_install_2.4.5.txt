
# Check prerequisites:
- Java JDK (1.8) // Check for Java 11 in Spark 3
- Hadoop binaries - /usr/local/hadoop

Installation
1 - Download Spark tar file
wget https://apache.mirrors.lucidnetworks.net/spark/spark-2.4.5/spark-2.4.5-bin-hadoop2.7.tgz


2 - Extract archive to /opt
sudo tar -xvzf -C /opt
